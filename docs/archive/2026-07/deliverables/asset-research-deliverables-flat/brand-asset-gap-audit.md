## Identity Implications

The current PNG is enough for the running prototype, but not enough for a durable GTG asset system. The top-ranked Customer Card System and GTG Topology SVG Kit both need an approved SVG source or a documented angular red motif extracted from the logo.

Provisional observed accent:

- Approximate PNG-sampled red: `#E90207`
- Source: sampled from `public/brand/gtg-logo.png`
- Status: reference only, not an approved official brand value

## Recommended Preparation

Prepare:

1. Official primary GTG logo SVG.
2. Inverse logo SVG for dark backgrounds.
3. Single-color mark SVG.
4. Favicon SVG.
5. Apple touch icon PNG.
6. Exact official red accent value.
7. Logo clear-space and minimum-size rule.

Do not:

- Trace or reinterpret the logo from a screenshot if an official vector exists.
- Use the GTG red motif in a way that makes customer logos look co-branded.
- Derive new facts, slogans, certifications, or partner claims from the logo system.
