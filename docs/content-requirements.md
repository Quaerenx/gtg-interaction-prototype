## Visual Asset Requirements

Hero assets:

- 7 custom wide images.
- Approximate ratio: 2.2:1.
- Should read well as perspective cards.
- Must not use OVA assets or visually duplicate the reference frames.

Solution assets:

- 5 custom full-bleed images.
- Ratio: 16:9 source minimum, with safe crop for desktop and mobile.
- Must support readable text overlays.
- Need light/dark variants or overlay rules for contrast.

Fallback assets:

- Static Hero background image.
- Static thumbnail or card image for each Hero item.
- Full-screen images reused for non-WebGL solutions.
